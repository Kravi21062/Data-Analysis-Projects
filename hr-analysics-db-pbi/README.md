# HR Analytics Dashboard (Power BI + Excel)

## Project Overview
This HR Analytics Dashboard helps track employee insights such as attrition, salary, performance, and hiring trends.

## Tools Used
- Power BI
- MS Excel
- DAX
- Power Query

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate %
- Average Salary
- Average Age
- Average Tenure
- Hiring (Last 12 Months)

## Visuals Included
- Attrition by Department
- Attrition by Job Role
- Attrition by Gender
- Attrition by Age Group
- Attrition by Salary Slab
- Attrition by Tenure
- Avg Salary by Department
- Performance Rating Distribution
- Monthly Hiring Trend
- New Joiners by Department

## Dataset
- 2000 rows synthetic HR dataset
- Columns: EmployeeID, Department, JobRole, Salary, Tenure, Attrition, etc.

## How to Use
1. Download the dataset and PBIX file
2. Open the `.pbix` file in Power BI Desktop
3. Refresh data if needed
4. Use slicers to filter department, role, gender, age group, education
